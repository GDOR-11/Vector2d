# Vector2d
a simple library for 2d vectors
<br>
<br>
<h1>properties:</h1>
<h4>x</h4>
the x coordinate
<h4>y</h4>
the y coordinate
<h4>length</h4>
length of the vector
<h4>angle</h4>
angle that the vector makes with the x-axis
<h1>methods:</h1>
<h2>static:</h2>
<h4>random()</h4>
returns a new random Vector with range x: \[0, 1) y: \[0, 1)
<h2>non-static:</h2>
<h4>add(vector)</h4>
receives as an argument
